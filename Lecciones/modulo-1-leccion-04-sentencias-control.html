Comparaciones

Nuestra primera clase aprendimos que eran los operadores condicionales, los cules incluan ==, !=, <=, >=, etc. Estos operadores son importantes en los bucles if de Python porque permiten que el programa tome decisiones basadas en una condicin. En otras palabras, los operadores condicionales permiten que el programa ejecute diferentes secciones de cdigo segn se cumpla o no una condicin dada.

En Python, el operador condicional ms comnmente utilizado en los bucles if es el operador de comparacin "==", que se utiliza para comparar dos valores. Por ejemplo, si queremos comprobar si una variable "x" es igual a 10. Recordemos algunos de ellos:

Qu es el control flow o control de flujo?

En Python, el "control flow" o control de flujo se refiere al orden en que se ejecutan las instrucciones en un programa o cdigo. El control flow determina qu instrucciones se ejecutan y en qu orden, y esto es controlado por estructuras como los bucles, las condiciones y las funciones.

Existen tres tipos principales de estructuras de control flow en Python:

  • Estructuras de seleccin: permiten al programa tomar decisiones y ejecutar diferentes secciones de cdigo en funcin de una condicin. La estructura de seleccin ms comn en Python es el bucle if, que ejecuta una seccin de cdigo si se cumple una condicin y otra seccin de cdigo si no se cumple la condicin.

    - if 
  - if ... else ...
  - if ... elif ... else

  • Estructuras de iteracin: permiten al programa repetir una seccin de cdigo varias veces. El bucle "for" es una estructura de iteracin comn en Python, que ejecuta una seccin de cdigo una vez para cada elemento de una lista o un rango de valores.

    - for
- while

  • Estructuras de control de funciones: permiten al programa llamar a una funcin y controlar el flujo de ejecucin del programa dentro y fuera de la funcin. Por ejemplo, la estructura "try/except" permite al programa manejar errores dentro de una funcin y continuar ejecutando el cdigo.

    - break
- continue
- pass

En resumen, el control flow en Python se refiere al orden en que se ejecutan las instrucciones en un programa y est controlado por estructuras como los bucles, las condiciones y las funciones. Estas estructuras permiten al programa tomar decisiones, repetir secciones de cdigo y controlar el flujo de ejecucin.

If ... else ...

Hemos visto que pasaba si la condicin que le pasabamos al if se cumpla, pero hasta ahora no hemos visto que pasaba si la condicin no se cumpla. En Python, el if...else es una estructura condicional que se utiliza para ejecutar un bloque de cdigo si se cumple una condicin y otro bloque de cdigo si no se cumple.

La sintaxis bsica del if...else en Python es la siguiente:

if condicion_1:
    # cdigo a ejecutar si se cumple la condicin 1
else:
    # cdigo a ejecutar si no se cumple ninguna de las condiciones anteriores

If ... elif ... else

Muchas veces, nos interesa chequear ms de una condicin. Como vimos, la condicin que ponemos en la lnea de la if tiene un True o un False como resultado, y con el else incluimos todas las situaciones que no se cumplen en el if. La sentencia elif de Python permite realizar comprobaciones continuas despus de una sentencia if inicial. Una sentencia elif se diferencia de la sentencia else porque se proporciona otra expresin/condicin para ser comprobada, al igual que con la sentencia if inicial.

Ya hemos visto el if ... "else, que como hemos dicho es una estructura condicional que se utiliza para ejecutar un bloque de cdigo si se cumple una condicin y otro bloque de cdigo si no se cumple, que segua la siguiente sintaxis:

if condicion:
    # cdigo a ejecutar si se cumple la condicin
else:
    # cdigo a ejecutar si no se cumple la condicin

En este ejemplo, "condicion" es una expresin booleana que se evala como verdadera o falsa. Si "condicion" es verdadera, se ejecutar el cdigo dentro del primer bloque if, y si es falsa, se ejecutar el cdigo dentro del bloque "else.

Adems, tambin se puede utilizar la estructura if else anidada para evaluar mltiples condiciones. En este caso, la sintaxis sera la siguiente:

if condicion_1:
    # cdigo a ejecutar si se cumple la condicin 1
elif condicion_2:
    # cdigo a ejecutar si no se cumple la condicin 1 pero se cumple la condicin 2
else:
    # cdigo a ejecutar si no se cumple ninguna de las condiciones anteriores

Imaginemos ahora que queremos incluir varias condiciones, estas seran:

  • Si tenemos ms de 50 peras, devolveremos un mensaje de que tenemos demasiadas peras y deberamos donarlas.

  • Si tenemos ms de 4 peras, devolveremos un mensaje diciendo que si no te apetece comer peras

  • Si tenemos ms de 0 peras y menos de 4, las apuntaremos en la lista de compra

  • Si no se cumple ninguna de las condiciones, devolveremos un mensaje de que nos hemos quedado sin peras.

If anidados

Bsicamente es una sentencia if dentro de otra sentencia if. Los if anidados pueden anidarse tanto como sea necesario, lo que permite construir programas muy complejos que responden a mltiples condiciones. Sin embargo, es importante tener en cuenta que el uso excesivo de if anidados puede hacer que el cdigo sea difcil de leer y mantener. Por lo tanto, es recomendable utilizarlos con moderacin y, en su lugar, utilizar tcnicas de programacin ms avanzadas, como las estructuras de datos y los algoritmos.

En este caso vamos a dividir nuestro cdigo en varios bloques:

    • 2.2 En caso de que no se cumpla la condicin anterior devolveremos un mensaje de que estamos sin peras.

Estructuras de iteracin

while

En Python, los bucles while son una estructura de control que permite ejecutar un bloque de cdigo varias veces mientras se cumpla una determinada condicin. El bucle while se repite continuamente mientras la condicin especificada sea verdadera. Si la condicin es falsa, el bucle while termina y la ejecucin contina con el cdigo que sigue al bucle.

La sintaxis bsica del bucle while en Python es la siguiente:

while condicin:
    # bloque de cdigo a ejecutar mientras se cumpla la condicin

La condicin en el bucle "while" puede ser cualquier expresin booleana, es decir, cualquier expresin que se evale como verdadera o falsa. Por ejemplo, podemos utilizar una variable, una comparacin, o cualquier otro operador o funcin que devuelva un valor booleano.

NORMALMENTE USAMOS UN while cuando no sabemos cuntas veces vamos a tener que ejecutar el bucle.

Es importante tener en cuenta que, si la condicin del bucle "while" nunca se vuelve falsa, el bucle se ejecutar infinitamente, lo que puede llevar a errores y problemas en el programa. Por lo tanto, es importante asegurarse de que la condicin se vuelve falsa en algn momento.
